# tests/ - Notebooks de Testing

Jupyter notebooks para validar los distintos componentes del proyecto.

## Notebooks

| Notebook | Que prueba |
|----------|-----------|
| `test_architecture.ipynb` | Validacion del diseno de arquitecturas CNN |
| `test_engine.ipynb` | Funcionalidad del motor de juego Quartopy |
| `test_engine_CNN.ipynb` | Integracion CNN con el motor de juego |
| `test_model.ipynb` | Carga e inferencia de modelos |
| `test_pred_francis.ipynb` | Predicciones de la arquitectura Francis/Extended |
| `test_Quarto.ipynb` | Reglas y logica del juego Quarto |
| `test_RL.ipynb` | Mecanicas de RL y entrenamiento basico |
| `test_RL2.ipynb` | Pruebas avanzadas del pipeline RL |
| `test_temperature.ipynb` | Efecto del parametro de temperatura en la seleccion de acciones |

## Uso
Abrir y ejecutar los notebooks en Jupyter para verificar que los componentes funcionan correctamente.
