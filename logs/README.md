# logs/ - Archivos de Log

Almacena logs de entrenamiento generados automaticamente por `runpy.sh` y los scripts de entrenamiento.

## Formato
Los archivos de log incluyen timestamps y mensajes coloreados del logger "TrainRL".
Se generan automaticamente al ejecutar entrenamientos via `runpy.sh`.

## Nota
Los logs de experimentos especificos se encuentran en sus carpetas de analisis correspondientes dentro de `analysis/`.
