# 02_includingBT/ - Desarrollo del Modelo Bradley-Terry

Proceso de prototipado e integracion del algoritmo de ranking Bradley-Terry al proyecto.

## Archivos
- `BT_imp.py`: Implementacion prototipo del modelo Bradley-Terry
- `test_bradley_terry.ipynb`: Test inicial del algoritmo
- `test_bradley_terry_full.ipynb`: Test completo
- `test_bradley_terry_full_xyz_true.ipynb`: Test con datos reales
- `test_BT_fcn_iter.ipynb`: Test de la funcion iterativa de convergencia
- `EXP_id03.pkl`: Datos del experimento EXP_id03 para pruebas
- `ploting.m`, `plotting_exp_xyz.m`: Scripts de MatLab para visualizacion

## Nota
La version final del algoritmo esta en `utils/metrics/bradley_terry.py`.
