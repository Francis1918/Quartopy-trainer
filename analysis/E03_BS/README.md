# E03_BS/ - Impacto del Batch Size

Estudio del efecto del batch size en el entrenamiento DQN.

## Variantes probadas
- Batch sizes: 32, 64, 512, 1024, 2048

## Notebooks
- `01_E03_whole_analysis.ipynb`: Analisis completo de todas las variantes

## Datos
- `E03_BATCH_SIZE*.pkl`: Resultados de entrenamiento por batch size
- `E03_BATCH_SIZE*.log`: Logs de entrenamiento
- `swiss_BATCH_SIZE*.log`: Logs de torneos suizos
- `swiss_tournament_*.pkl`: Resultados de torneos
