# E08_loss-balance/ - Balance de Funciones de Loss

Comparacion de las tres funciones de loss disponibles usando torneos suizos.

## Loss Approaches Comparados
- `combined_avg`: Promedio de Q-values de colocacion y seleccion
- `only_select`: Solo Q-values de seleccion de pieza
- `only_place`: Solo Q-values de colocacion en tablero
